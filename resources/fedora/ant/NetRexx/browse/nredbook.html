NetRexx Redbook

Redbooks, named for their red covers, are "how to" books, written by very experienced IBM professionals from all over the world. They are written at IBM's International Technical Support Organization (ITSO), which has centers close to IBM's development divisions.

The NetRexx Redbook Creating Java Applications with NetRexx, SG24-2216 can now be ordered from IBM, either from your local IBM Branch or from the IBM Redbooks home page.

It is also available online on the IBM ITSO Redbook home page: 

http://www.redbooks.ibm.com
Search for SG24-2216 and you should find pointers to the abstract and the PDF file, or you can find the PDF file directly here: Creating Java Applications using NetRexx.

The redbook samples file, nrxredbk.zip, is available at 

ftp://www.redbooks.ibm.com/redbooks/SG242216

The examples were developed and tested on Windows and OS/2, based on JDK 1.1.1. There is currently [4 Sep 1997] one known problem on OS/2: thread termination in one sample gives a null pointer exception in the stop method.

The GUI examples need a JDK 1.1.1 enabled browser, or appletviewer, for example, Netscape 4.02 for Windows with an additional zip file ('JDK 1.1 patch') from the Netscape download site runs now JDK 1.1.1
